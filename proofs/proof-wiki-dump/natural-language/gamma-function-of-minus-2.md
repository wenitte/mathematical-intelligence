Theorem
$\map \Gamma {-2}$ is not defined.
where $\Gamma$ denotes the Gamma function.


Proof













\(\ds \map \Gamma {-1}\)

\(=\)







\(\ds \paren {-2} \, \map \Gamma {-2}\)





Gamma Difference Equation








\(\ds \leadsto \ \ \)





\(\ds \map \Gamma {-2}\)

\(=\)







\(\ds \dfrac {\map \Gamma {-1} } {-2}\)









But from Gamma Function of Minus 1, $\map \Gamma {-1}$ is not defined.
Hence the result.
$\blacksquare$
