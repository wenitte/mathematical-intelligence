Theorem
$\dfrac {3 + 25 + 38} {7 + 20 + 39} = \dfrac {3^4 + 25^4 + 38^4} {7^4 + 20^4 + 39^4}$


Proof













\(\ds \dfrac {3^4 + 25^4 + 38^4} {7^4 + 20^4 + 39^4}\)

\(=\)







\(\ds \dfrac {81 + 390625 + 2085136} {2401 + 160000 + 2313441}\)




















\(\ds \)

\(=\)







\(\ds \dfrac {2475842} {2475842}\)
























\(\ds \dfrac {3 + 25 + 38} {7 + 20 + 39}\)

\(=\)







\(\ds \dfrac {66} {66}\)




















\(\ds \)

\(=\)







\(\ds \dfrac {2475842} {2475842}\)









$\blacksquare$
