Theorem
The complement of the universal set is the empty set:

$\map \complement {\mathbb U} = \O$


Proof
Substitute $\mathbb U$ for $S$ in $\relcomp S S = \O$ from Relative Complement with Self is Empty Set.
$\blacksquare$
