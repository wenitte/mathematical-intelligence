Example of Derivative of Function of Constant Multiple
$\map {\dfrac \d {\d x} } {\map \sec {a x + b} } = a \map \sec {a x + b} \map \tan {a x + b}$


Proof













\(\ds \map {\dfrac \d {\d x} } {\map \sec {a x + b} }\)

\(=\)







\(\ds a \map {\dfrac \d {\map \d {a x + b} } } {\map \sec {a x + b} }\)





Derivative of Function of Constant Multiple: Corollary














\(\ds \)

\(=\)







\(\ds a \map \sec {a x + b} \map \tan {a x + b}\)





Derivative of Secant Function



$\blacksquare$
