Theorem
The intersection of a set with the universal set is the set itself:

$\mathbb U \cap S = S$


Proof













\(\ds S\)

\(\subseteq\)







\(\ds \mathbb U\)





Definition of Universal Set








\(\ds \leadstoandfrom \ \ \)





\(\ds \mathbb U \cap S\)

\(=\)







\(\ds S\)





Intersection with Subset is Subset‎



$\blacksquare$
