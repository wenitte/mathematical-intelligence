Theorem
$\forall x \in \R: 0 \times x = 0$


Proof













\(\ds 0 \times x\)

\(=\)







\(\ds \paren {0 + 0} \times x\)





Real Number Axiom $\R \text A3$: Identity for Addition














\(\ds \)

\(=\)







\(\ds 0 \times x + 0 \times x\)





Real Number Axiom $\R \text D$: Distributivity of Multiplication over Addition








\(\ds \leadsto \ \ \)





\(\ds 0 \times x\)

\(=\)







\(\ds 0\)





Real Addition Identity is Zero: Corollary



$\blacksquare$
