Example of Derivative of Function of Constant Multiple
$\map {\dfrac \d {\d x} } {\map \cos {a x + b} } = -a \map \sin {a x + b}$


Proof













\(\ds \map {\dfrac \d {\d x} } {\map \cos {a x + b} }\)

\(=\)







\(\ds a \map {\dfrac \d {\map \d {a x + b} } } {\map \cos {a x + b} }\)





Derivative of Function of Constant Multiple: Corollary














\(\ds \)

\(=\)







\(\ds -a \map \sin {a x + b}\)





Derivative of Cosine Function



$\blacksquare$
