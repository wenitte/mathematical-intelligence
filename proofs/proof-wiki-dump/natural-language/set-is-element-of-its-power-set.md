Theorem
A set is an element of its power set:

$S \in \powerset S$


Proof









\(\ds \forall S: \, \)



\(\ds S\)

\(\subseteq\)







\(\ds S\)





Set is Subset of Itself








\(\ds \leadsto \ \ \)

\(\ds \forall S: \, \)



\(\ds S\)

\(\in\)







\(\ds \powerset S\)





Definition of Power Set



$\blacksquare$
