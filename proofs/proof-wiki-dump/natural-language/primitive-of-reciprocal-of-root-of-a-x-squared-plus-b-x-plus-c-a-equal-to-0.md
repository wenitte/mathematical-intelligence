Theorem
For $a x^2 + b x + c > 0$:

$\ds \int \frac {\d x} {\sqrt {a x^2 + b x + c} } = \frac {2 \sqrt {b x + c} } b + C$
when $a = 0$.


Proof













\(\ds a\)

\(=\)







\(\ds 0\)














\(\ds \leadsto \ \ \)





\(\ds \int \frac {\d x} {\sqrt {a x^2 + b x + c} }\)

\(=\)







\(\ds \int \frac {\d x} {\sqrt {b x + c} }\)




















\(\ds \)

\(=\)







\(\ds \frac {2 \sqrt {b x + c} } b\)





Primitive of $\dfrac 1 {\sqrt {a x + b} }$



$\blacksquare$
