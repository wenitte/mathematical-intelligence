Theorem
Let $\coth: \C \to \C$ be the hyperbolic cotangent function on the set of complex numbers.

Then $\coth$ is odd:

$\map \coth {-x} = -\coth x$


Proof













\(\ds \map \coth {-x}\)

\(=\)







\(\ds \frac {\map \cosh {-x} } {\map \sinh {-x} }\)





Definition 2 of Hyperbolic Cotangent














\(\ds \)

\(=\)







\(\ds \frac {\map \cosh {-x} }  {-\sinh x}\)





Hyperbolic Sine Function is Odd














\(\ds \)

\(=\)







\(\ds \frac {\cosh x} {-\sinh x}\)





Hyperbolic Cosine Function is Even














\(\ds \)

\(=\)







\(\ds -\coth x\)









$\blacksquare$
