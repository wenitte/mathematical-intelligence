Example of Fibonacci Strings
The Fibonacci string $S_4$ is $\text{bab}$.


Proof













\(\ds S_4\)

\(=\)







\(\ds S_3 S_2\)





Definition of Fibonacci String














\(\ds \)

\(=\)







\(\ds \text{ba} \ \text{b}\)





Definition of Fibonacci String $S_3$



$\blacksquare$
