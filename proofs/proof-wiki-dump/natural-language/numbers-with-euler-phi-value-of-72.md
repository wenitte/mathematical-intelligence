Example of Use of Euler $\phi$ Function
There are $17$  positive integers for which the value of the Euler $\phi$ function is $72$:

$73, 91, 95, 111, 117, 135, 146, 148, 152, 182, 190, 216, 222, 228, 234, 252, 270$


Proof













\(\ds 72\)

\(=\)







\(\ds \map \phi {73}\)
