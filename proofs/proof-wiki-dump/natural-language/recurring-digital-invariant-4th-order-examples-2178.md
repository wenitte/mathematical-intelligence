Example of $4$th Order Recurring Digital Invariant
$2178$ is a $4$th order recurring digital invariant:








\(\ds 2178: \ \ \)





\(\ds 2^4 + 1^4 + 7^4 + 8^4\)

\(=\)







\(\ds 16 + 1 + 2401 + 4096\)

\(\ds = 6514\)












\(\ds 6514: \ \ \)





\(\ds 6^4 + 5^4 + 1^4 + 4^4\)

\(=\)







\(\ds 1296 + 625 + 1 + 256\)

\(\ds = 2178\)







$\blacksquare$
