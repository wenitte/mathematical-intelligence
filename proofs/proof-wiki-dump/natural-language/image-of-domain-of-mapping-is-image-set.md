Theorem
Let $S$ and $T$ be sets.
Let $f: S \to T$ be a mapping.

The image of $S$ is the image set of $f$:

$f \sqbrk S = \Img f$


Proof
By definition, a mapping is a relation.
Thus Image of Domain of Relation is Image Set applies.
$\blacksquare$
