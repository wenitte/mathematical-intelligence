Theorem
The relative complement of a set in itself is the empty set:

$\relcomp S S = \O$


Proof













\(\ds \relcomp S S\)

\(=\)







\(\ds S \setminus S\)





Definition of Relative Complement














\(\ds \)

\(=\)







\(\ds \O\)





Set Difference with Self is Empty Set



$\blacksquare$
