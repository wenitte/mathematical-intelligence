Theorem
Let $a, b, c, d \in \R$ such that $a > b$ and $c > d$.

Then:

$a + c > b + d$


Proof 1













\(\ds a\)

\(>\)







\(\ds b\)














\(\ds \leadsto \ \ \)





\(\ds a + c\)

\(>\)







\(\ds b + c\)





Real Number Ordering is Compatible with Addition


















\(\ds c\)

\(>\)







\(\ds d\)














\(\ds \leadsto \ \ \)





\(\ds b + c\)

\(>\)







\(\ds b + d\)





Real Number Ordering is Compatible with Addition




Finally:














\(\ds a + c\)

\(>\)







\(\ds b + c\)




















\(\ds b + c\)

\(>\)







\(\ds b + d\)














\(\ds \leadsto \ \ \)





\(\ds a + c\)

\(>\)







\(\ds b + d\)





Trichotomy Law for Real Numbers



$\blacksquare$
