Example of Logarithmic Differentiation
Let $y = x \paren {1 + 2 x} \paren {1 + 3 x}$.
Then:

$\dfrac {\d y} {\d x} = \paren {1 + 2 x} \paren {1 + 3 x} + 2 x \paren {1 + 3 x} + 3 x \paren {1 + 2 x}$


Proof













\(\ds y\)

\(=\)







\(\ds x \paren {1 + 2 x} \paren {1 + 3 x}\)














\(\ds \leadsto \ \ \)





\(\ds \ln y\)

\(=\)







\(\ds \map \ln {x \paren {1 + 2 x} \paren {1 + 3 x} }\)




















\(\ds \)

\(=\)







\(\ds \map \ln x + \map \ln {1 + 2 x} + \map \ln {1 + 3 x}\)





Sum of Logarithms








\(\ds \leadsto \ \ \)





\(\ds \dfrac 1 y \dfrac {\d y} {\d x}\)

\(=\)







\(\ds \dfrac 1 x + \dfrac 2 {1 + 2 x} + \dfrac 3 {1 + 3 x}\)





Chain Rule for Derivatives, Derivative of Natural Logarithm, Derivative of Constant Multiple








\(\ds \leadsto \ \ \)





\(\ds \dfrac {\d y} {\d x}\)

\(=\)







\(\ds y \paren {\dfrac 1 x + \dfrac 2 {1 + 2 x} + \dfrac 3 {1 + 3 x} }\)





simplifying














\(\ds \)

\(=\)







\(\ds \paren {x \paren {1 + 2 x} \paren {1 + 3 x} } \paren {\dfrac 1 x + \dfrac 2 {1 + 2 x} + \dfrac 3 {1 + 3 x} }\)





by hypothesis














\(\ds \)

\(=\)







\(\ds \paren {1 + 2 x} \paren {1 + 3 x} + 2 x \paren {1 + 3 x} + 3 x \paren {1 + 2 x}\)





simplification



$\blacksquare$
