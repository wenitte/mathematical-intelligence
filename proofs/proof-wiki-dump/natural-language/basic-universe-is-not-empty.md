Theorem
Let $V$ be a basic universe
Then $V$ is not the empty class.


Proof
The Axiom of the Empty Set gives us:
The empty class $\O$ is a set, that is:

$\O \in V$

Hence the result by definition of empty class.
$\blacksquare$
