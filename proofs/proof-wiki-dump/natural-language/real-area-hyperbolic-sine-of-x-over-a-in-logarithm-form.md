Theorem













\(\ds \arsinh \dfrac x a\)

\(=\)







\(\ds \map \ln {\frac {x + \sqrt {x^2 + a^2} } a}\)




















\(\ds \)

\(=\)







\(\ds \map \ln {x + \sqrt {x^2 + a^2} } - \ln a\)











Proof













\(\ds \arsinh \frac x a\)

\(=\)







\(\ds \map \ln {\frac x a + \sqrt {\paren {\frac x a}^2 + 1} }\)
