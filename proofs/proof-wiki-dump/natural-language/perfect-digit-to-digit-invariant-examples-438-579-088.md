Example of Perfect Digit-to-Digit Invariant
$438 \, 579 \, 088$ is a perfect digit-to-digit invariant:

$438 \, 579 \, 088 = 4^4 + 3^3 + 8^8 + 5^5 + 7^7 + 9^9 + 0^0 + 8^8 + 8^8$


Proof













\(\ds \)

\(\)







\(\ds 4^4 + 3^3 + 8^8 + 5^5 + 7^7 + 9^9 + 0^0 + 8^8 + 8^8\)




















\(\ds \)

\(=\)







\(\ds 256 + 27 + 16 \, 777 \, 216 + 3125 + 823 \, 543 + 387 \, 420 \, 489 + 0 + 16 \, 777 \, 216 + 16 \, 777 \, 216\)




















\(\ds \)

\(=\)







\(\ds 438 \, 579 \, 088\)









$\blacksquare$
