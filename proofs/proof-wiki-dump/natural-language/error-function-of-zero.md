Theorem
$\map \erf 0 = 0$
where $\erf$ denotes the error function. 


Proof
By Error Function is Odd, $\erf$ is a odd function. 
Therefore, by Odd Function of Zero is Zero:

$\map \erf 0 = 0$
$\blacksquare$
