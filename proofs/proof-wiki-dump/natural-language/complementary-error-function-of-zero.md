Theorem
$\map \erfc 0 = 1$
where $\erfc$ denotes the complementary error function.


Proof













\(\ds \map \erfc 0\)

\(=\)







\(\ds 1 - \map \erf 0\)





Definition of Complementary Error Function














\(\ds \)

\(=\)







\(\ds 1\)





Error Function of Zero



$\blacksquare$
