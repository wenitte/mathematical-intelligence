Theorem
The class of finite models is not $\Delta$-elementary.
That is:

there is no set of formulas $F$ such that $F$ is satisfied by a model $\MM$
if and only if:

$\MM$ is finite.


Proof
Follows directly from the Overflow Theorem.
$\blacksquare$
