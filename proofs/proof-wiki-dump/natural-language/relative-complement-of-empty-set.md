Theorem
The relative complement of the empty set is the set itself:

$\relcomp S \O = S$


Proof













\(\ds \relcomp S \O\)

\(=\)







\(\ds S \setminus \O\)





Definition of Relative Complement














\(\ds \)

\(=\)







\(\ds S\)





Set Difference with Empty Set is Self



$\blacksquare$
