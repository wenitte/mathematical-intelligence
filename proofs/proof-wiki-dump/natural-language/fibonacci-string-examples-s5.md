Example of Fibonacci Strings
The Fibonacci string $S_5$ is $\text{babba}$.


Proof













\(\ds S_5\)

\(=\)







\(\ds S_4 S_3\)





Definition of Fibonacci String














\(\ds \)

\(=\)







\(\ds \text{bab} \ \text{ba}\)





Fibonacci String $S_4$ and Fibonacci String $S_3$



$\blacksquare$
