Examples of Cotangent Function is Odd
$\map \cot {-2 x} = -\cot 2 x$


Proof
From Cotangent Function is Odd:

$\map \cot {-\theta} = -\cot \theta$
The result follows by setting $\theta = 2 x$.
$\blacksquare$
