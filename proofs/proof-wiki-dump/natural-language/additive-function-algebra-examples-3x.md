Example of Additive Function
Let $\map f x$ be the real function defined as:

$\forall x \in \R: \map f x = 3 x$
Then $f$ is an additive function.


Proof













\(\ds \map f {x + y}\)

\(=\)







\(\ds 3 \paren {x + y}\)




















\(\ds \)

\(=\)







\(\ds 3 x + 3 y\)




















\(\ds \)

\(=\)







\(\ds \map f x + \map f y\)









$\blacksquare$
