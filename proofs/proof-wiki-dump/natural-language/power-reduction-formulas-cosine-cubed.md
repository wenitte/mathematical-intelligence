Theorem
$\cos^3 x = \dfrac {3 \cos x + \cos 3 x} 4$
where $\cos$ denotes cosine.


Proof













\(\ds \cos 3 x\)

\(=\)







\(\ds 4 \cos^3 x - 3 \cos x\)





Triple Angle Formula for Cosine








\(\ds \leadsto \ \ \)





\(\ds 4 \cos^3 x\)

\(=\)







\(\ds 3 \cos x + \cos 3 x\)





rearranging








\(\ds \leadsto \ \ \)





\(\ds \cos^3 x\)

\(=\)







\(\ds \dfrac {3 \cos x + \cos 3 x} 4\)





dividing both sides by $4$



$\blacksquare$
