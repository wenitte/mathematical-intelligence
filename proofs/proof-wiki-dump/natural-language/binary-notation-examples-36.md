Example of Binary Notation
The number written in decimal notation as $36$ is expressed in binary notation as $100100_2$.


Proof













\(\ds 36\)

\(=\)







\(\ds 1 \times 2^5 + 0 \times 2^4 + 0 \times 2^3 + 1 \times 2^2 + 0 \times 2^1 + 0 \times 2^0\)




















\(\ds \)

\(=\)







\(\ds 32 + 0 + 0 + 4 + 0 + 0\)




















\(\ds \)

\(=\)







\(\ds \sqbrk {100100}_2\)









$\blacksquare$
