Theorem
$-100 \bmod 0 = -100$
where $\bmod$ denotes the modulo operation.


Proof
By definition of modulo $0$:

$\forall x \in \R: x \bmod 0 = x$
Hence:

$-100 \bmod 0 = -100$
$\blacksquare$
