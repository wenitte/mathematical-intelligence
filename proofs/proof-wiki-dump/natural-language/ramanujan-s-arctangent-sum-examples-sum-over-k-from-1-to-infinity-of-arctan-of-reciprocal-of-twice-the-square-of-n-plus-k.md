Example of Use of Ramanujan's Arctangent Sum
$\ds \sum_{k \mathop = 1}^\infty \map \arctan {\dfrac 1 {2 \paren {n + k}^2} } = \map \arctan {\dfrac 1 {2 n + 1} } $


Proof













\(\ds \sum_{k \mathop = 0}^\infty \map \arctan {\dfrac 2 {\paren {n + 2 k + 1}^2} }\)

\(=\)







\(\ds \map \arctan {\dfrac 1 n }\)





Ramanujan's Arctangent Sum: Corollary








\(\ds \leadsto \ \ \)





\(\ds \sum_{k \mathop = 0}^\infty \map \arctan {\dfrac 2 {\paren {\paren {2 n + 1} + 2 k + 1}^2} }\)

\(=\)







\(\ds \map \arctan {\dfrac 1 {2 n + 1} }\)





replacing $n$ with $2 n + 1$








\(\ds \leadsto \ \ \)





\(\ds \sum_{k \mathop = 0}^\infty \map \arctan {\dfrac 2 {4 \paren {n + k + 1}^2} }\)

\(=\)







\(\ds \map \arctan {\dfrac 1 {2 n + 1} }\)














\(\ds \leadsto \ \ \)





\(\ds \sum_{k \mathop = 0}^\infty \map \arctan {\dfrac 1 {2 \paren {n + k + 1}^2} }\)

\(=\)







\(\ds \map \arctan {\dfrac 1 {2 n + 1} }\)














\(\ds \leadsto \ \ \)





\(\ds \sum_{k \mathop = 1}^\infty \map \arctan {\dfrac 1 {2 \paren {n + k}^2} }\)

\(=\)







\(\ds \map \arctan {\dfrac 1 {2 n + 1} }\)





reindexing the sum



$\blacksquare$
