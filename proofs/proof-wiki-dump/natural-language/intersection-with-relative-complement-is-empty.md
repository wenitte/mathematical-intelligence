Theorem
The intersection of a set and its relative complement is the empty set:

$T \cap \relcomp S T = \O$


Proof













\(\ds T \cap \relcomp S T\)

\(=\)







\(\ds \paren {S \setminus T} \cap T\)





Definition of Relative Complement














\(\ds \)

\(=\)







\(\ds \O\)





Set Difference Intersection with Second Set is Empty Set



$\blacksquare$
