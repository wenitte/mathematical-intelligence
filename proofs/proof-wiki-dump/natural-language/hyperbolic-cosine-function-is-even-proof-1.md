Theorem
$\map \cosh {-x} = \cosh x$


Proof













\(\ds \map \cosh {-x}\)

\(=\)







\(\ds \frac {e^{-x} + e^{-\paren {-x} } } 2\)





Definition of Hyperbolic Cosine














\(\ds \)

\(=\)







\(\ds \frac {e^{-x} + e^x} 2\)




















\(\ds \)

\(=\)







\(\ds \frac {e^x + e^{-x} } 2\)




















\(\ds \)

\(=\)







\(\ds \cosh x\)









$\blacksquare$
