Theorem
$\ds \int \frac {\d x} {x \sqrt {x^2 + a^2} } = \frac 1 a \map \ln {\frac x {a + \sqrt {x^2 + a^2} } } + C$


Proof













\(\ds \int \frac {\d x} {x \sqrt {x^2 + a^2} }\)

\(=\)







\(\ds -\frac 1 a \map \ln {\frac {a + \sqrt {x^2 + a^2} } x} + C\)





Primitive of Reciprocal of $x \sqrt {x^2 + a^2}$: Logarithm form














\(\ds \)

\(=\)







\(\ds \frac 1 a \map \ln {\frac x {a + \sqrt {a^2 + x^2} } } + C\)





Logarithm of Reciprocal



$\blacksquare$
