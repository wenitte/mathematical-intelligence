Theorem
Let $\RR$ be an equivalence relation on a set $S$.
Then no $\RR$-class is empty.


Proof









\(\ds \forall \eqclass x \RR \subseteq S: \exists x \in S: \, \)



\(\ds x\)

\(\in\)







\(\ds \eqclass x \RR\)





Definition of Equivalence Class








\(\ds \leadsto \ \ \)





\(\ds \eqclass x \RR\)

\(\ne\)







\(\ds \O\)





Definition of Empty Set




$\blacksquare$
