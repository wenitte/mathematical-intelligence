Example of Use of Divisor Count Function
$\map {\sigma_0} 3 = 2$
where $\sigma_0$ denotes the divisor count function.


Proof
We have that $3$ is a prime number.
From Divisor Count Function of Prime Number:

$\map {\sigma_0} 3 = 2$
$\blacksquare$
